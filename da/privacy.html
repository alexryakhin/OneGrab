Privatlivspolitik

Sidst opdateret: 22. februar 2026

Overblik

OneGrab («vi») respekterer dit privatliv. Denne politik beskriver, hvordan vi behandler oplysninger i forbindelse med OneGrab-appen og vores hjemmeside.

Data vi indsamler

OneGrab er designet til at minimere dataindsamling. Vi kan indsamle: analyse og brug (Firebase Analytics, anonyme begivenheder), nedbrud (Crashlytics), abonnementer og køb (Apple, RevenueCat), webanalyse.

Data vi ikke indsamler

Vi indsamler eller gemmer ikke de URL'er, du indsætter, det indhold du downloader eller din identitet. Downloads sker på din enhed; vi gemmer ikke kopier på vores servere.

Brug af data

Vi bruger disse data til at drive og forbedre appen, behandle abonnementer, rette nedbrud og overholde juridiske forpligtelser.

Tredjeparter

Vi bruger Apple, RevenueCat og Google Firebase. Deres privatlivspolitikker gælder. Vi sælger ikke dine data.

Dine rettigheder

Du kan anmode om adgang, rettelse eller sletning af dine data, hvor loven tillader det. Kontakt os via supportmuligheden i appen eller på hjemmesiden.

Ændringer

Vi kan opdatere denne politik. Datoen «Sidst opdateret» vil ændre sig. Fortsat brug udgør accept.
